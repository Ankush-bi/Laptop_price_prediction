About the project

Built to make laptop pricing easier to understand.

This project predicts laptop prices using machine learning algorithms and presents the result through a simple Flask web application. It brings together Python, HTML, and CSS to make the experience smooth and approachable.

Machine Learning

The app uses a trained regression model to estimate prices from laptop attributes.

Modern Web Design

Each page is designed to feel clean, responsive, and easy to navigate.

Practical Use

It is a useful example of applying data science concepts in a real-world web app.
